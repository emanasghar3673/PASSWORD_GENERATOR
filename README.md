# Password Generator

## Description

This is a simple Python project that generates a strong random password based on the user-defined length. It uses uppercase letters, lowercase letters, numbers, and special characters to create secure passwords.

## Features

- User-defined password length
- Random password generation
- Includes letters, digits, and special characters
- Simple and beginner-friendly Python code

## Technologies Used

- Python 3
- random module
- string module

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:

```bash
python password_generator.py
```

4. Enter the desired password length.
5. The program will generate a random password.

## Example

```
Enter the password length: 10

Generated Password: A#8dP@2mQ!
```
